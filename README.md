**Overview**

This project is an interactive Sales Analytics Dashboard built using Power BI, designed to enable data-driven decision-making by tracking sales performance, trends, and key business metrics.

**Objective:**
To Provide a single-view dashboard which enable Quick business decisions through visual analytics

**Key Features:**

1.KPIs for revenue, growth trends, and sales distribution

2.Interactive filters for time, region and product categories

3.Visual breakdown of sales performance across dimensions

**Tools Used:**

Power BI  

Data modeling & DAX    

Data visualization & dashboard design
